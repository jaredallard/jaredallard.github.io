About Me



I'm a 17 year old programmer, trying to create; useful, meaningful, and creative applications.

2008-2012

      In 2008, my coding life began, starting with batch and then later bash. I focused on these languages for about 3~ years, creating various low-level console applications. Not satisfied with the speeds of low-level scripting languages; I picked up PHP. There I started work on a few projects such as; PHC-WIN, rrpg, and PHC-LIN. Soon thereafter I found myself in need of after-page-load actions, so I picked up Javascript and fell inlove with it.

2013-2014

      After picking up Javascript I began working with third party CSS & JS librarys such as; bootstrap, font-awesome, jquery, and app.js. Using some of these technologies I started developing a Twitter client. I soon discovered pure js wouldn't work, that's when I found out about node.js. Loving the idea, I developed an API for an old school of mine, TCS-api which allows a mobile app too check-in/check-out of school and handle JC cases (the schools form of discipline). Since then I've been hard at work on Twimber and Equestria's Betrayal.

2015+

      Since the end of 2014 plus I've been looking for any work I can find! I'm now a maintainer of crcn/nexe, and done several projects including a Modern Survey Site! I'm always looking for fun and innovative projects to contribute too.
