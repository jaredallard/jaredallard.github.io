About Me



2008-2012

      I'm a 16 year old software developer. I've been programming since I was 10, starting with batch, and then later bash. I focused on these languages for about 3~ years, creating various low-level console applications. Not satisfied with the speeds of low-level scripting languages; I picked up PHP. There I started work on a few projects such as; PHC-WIN, rrpg, and PHC-LIN. Soon thereafter I found myself in need of after-page-load actions, so I picked up Javascript and fell inlove with it.

2013-2014

      After picking up Javascript I began working with third party CSS librarys such as; bootstrap, font-awesome, and JQuery-UI. Using some of these technologies I started developing a Twitter client. I soon discovered pure js wouldn't work, that's when I found out about node.js. Loving the idea, I developed an API for an old school of mine, TCS-api which allows a mobile app too check-in/check-out of school and handle JC cases (the schools form of discipline). Since then I've been hard at work on Twimber and Equestria's Betrayal.
